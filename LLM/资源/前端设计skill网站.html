UI / UX & Design 资源官网合集 ​

1. Anthropic（UI/产品体验相关公司） ​

https://www.anthropic.com

Claude 的官方公司

他们本身不专注 UI 框架，但产品交互设计很有参考价值

重点看 Claude UI / AI 交互范式

2. UI UX Pro Max（AI UI设计技能库） ​

https://ui-ux-pro-max-skill.com

AI UI设计"设计系统数据库"

包含：配色 / 字体 / 风格 / UX规则

很多 Cursor / Claude Code 用户在用

3. Impeccable UI（UI skill） ​

https://impeccable-ui.com

偏"AI UI设计规范库"

用于生成更干净的界面风格

和 UI UX Pro Max 属于同一类工具生态

4. UIUX Pro Max（官方文档/介绍） ​

https://uupm.cc

https://ui-ux-pro-max-skill.com

设计系统增强 AI 输出质量

强调：style + UX + typography + spacing

5. 0xDesign（设计灵感/前端设计社区） ​

https://0xdesign.dev

偏前端 + UI + 产品设计审美

很多 AI UI / Dev UI 风格参考来源

极简 + 工程师审美

6. User Interface Wiki（UI模式百科） ​

https://ui-patterns.com

UI设计模式百科全书

比如：modal / sidebar / dashboard pattern

非常适合系统学习 UI 结构

7. Designer Skills（设计能力学习体系） ​

https://www.nngroup.com

（Nielsen Norman Group，权威 UX 机构）

UX行业标准来源

可用性测试 / 信息架构 / 交互设计核心理论

8. Make Interfaces Better（UI改进理念） ​

https://lawsofux.com

UX 设计 10 大法则

比如：Hick's Law / Fitts's Law

非常适合提升"设计判断力"

9. UIUX Wiki / Design Wiki（通用 UI知识库） ​

https://www.uxdesign.cc

Medium 上最活跃的 UX 设计社区

UI趋势 / 案例 / 产品拆解

10. UIUX Pro / Design Systems（广义设计系统参考） ​

https://material.io

Google Material Design 官方

UI设计标准体系（颜色 / spacing / motion）
