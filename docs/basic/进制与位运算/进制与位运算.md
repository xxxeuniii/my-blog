# 进制与位运算

## 一、进制概述

### 1.1 常见进制

- **二进制（Binary）**：由 0 和 1 组成，逢二进一，基数是 2
- **八进制（Octal）**：由 0-7 组成，逢八进一，基数是 8
- **十进制（Decimal）**：由 0-9 组成，逢十进一，基数是 10
- **十六进制（Hexadecimal）**：由 0-9 和 A-F 组成，逢十六进一，基数是 16

### 1.2 进制转换

**十进制转其他进制**：除基取余，逆序余数

```
十进制 13 转二进制：13 ÷ 2 = 6 ... 1
6 ÷ 2 = 3 ... 0
3 ÷ 2 = 1 ... 1
1 ÷ 2 = 0 ... 1
结果：1101
```

**其他进制转十进制**：按权展开求和

```
二进制 1101 转十进制：
1×2³ + 1×2² + 0×2¹ + 1×2⁰ = 8 + 4 + 0 + 1 = 13
```

## 二、位运算

### 2.1 按位与（&）

对应位都为 1 时结果为 1

```
5 & 3 = 0101 & 0011 = 0001 = 1
```

### 2.2 按位或（|）

对应位有一个为 1 时结果就为 1

```
5 | 3 = 0101 | 0011 = 0111 = 7
```

### 2.3 按位异或（^）

对应位相同时为 0，不同时为 1

```
5 ^ 3 = 0101 ^ 0011 = 0110 = 6
```

**特性**：`a ^ a = 0`，`a ^ 0 = a`

### 2.4 移位运算

- **左移（<<）**：`n << m = n × 2^m`
- **右移（>>）**：`n >> m = n ÷ 2^m`

## 三、原码、反码、补码

- **原码**：符号位 + 绝对值的二进制
- **反码**：正数不变，负数其余位取反
- **补码**：正数不变，负数为反码 + 1

补码让计算机可以用加法实现减法。

## 四、位、字节与编码

- **位（bit）**：最小单位，0 或 1
- **字节（Byte）**：8 位

**编码**：
- ASCII：7 位，128 个字符
- Unicode：统一码
- UTF-8：可变长度，中文 3 字节
