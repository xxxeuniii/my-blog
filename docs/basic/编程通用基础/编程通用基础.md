# 编程通用基础

## 一、基础概念

### 变量与常量
- **变量**：可变的存储空间
- **常量**：不可变的值（const/final）

### 数据类型
- 基础类型：number、string、boolean
- 引用类型：object、array、function

### 作用域
- 全局作用域、函数作用域、块级作用域

## 二、面向对象

### 三大特性
- **封装**：隐藏实现细节
- **继承**：复用代码
- **多态**：不同对象同一行为不同结果

### 接口与抽象类
- **接口**：行为规范
- **抽象类**：部分实现，不可实例化

### 泛型
- 通用类型，不指定具体类型

### 异常处理
- try-catch-finally
- 抛出异常：throw

## 三、工程化

### 模块化
- CommonJS、ES Module
- 导出/导入

### 版本控制 Git
```bash
git init           # 初始化
git add            # 暂存
git commit         # 提交
git push/pull      # 远程同步
git branch         # 分支操作
```
