# 数据结构与算法

## 一、基础数据结构

### 数组（Array）
- 连续内存存储
- 随机访问 O(1)
- 插入/删除 O(n)

### 链表（Linked List）
- 节点通过指针连接
- 插入/删除 O(1)
- 访问 O(n)

### 栈（Stack）
- 先进后出（LIFO）
- 操作：push、pop

### 队列（Queue）
- 先进先出（FIFO）
- 操作：enqueue、dequeue

### 哈希表（Hash Table）
- 键值对存储
- 查找 O(1)
- 冲突解决：链地址法、开放地址法

### 二叉树
- 每个节点最多两个子节点
- 遍历：前序、中序、后序、层序

## 二、基础算法

### 排序
- 冒泡排序：O(n²)
- 快速排序：O(n log n)
- 归并排序：O(n log n)
- 堆排序：O(n log n)

### 二分查找
- 有序数组查找
- 时间复杂度 O(log n)

### 递归
- 函数自身调用
- 注意递归深度和base case

### 双指针
- 滑动窗口
- 快慢指针

## 三、复杂度

### 时间复杂度
- O(1) < O(log n) < O(n) < O(n log n) < O(n²)

### 空间复杂度
- 算法占用的额外内存空间
