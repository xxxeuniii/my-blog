# 算法

## 一、排序算法

### 冒泡排序
- 相邻元素比较交换
- 时间复杂度 O(n²)
- 空间复杂度 O(1)

### 快速排序
- 分治思想，选基准分区
- 平均时间复杂度 O(n log n)
- 最差 O(n²)

### 归并排序
- 分治思想，拆分后合并
- 时间复杂度 O(n log n)
- 空间复杂度 O(n)

### 堆排序
- 利用堆结构
- 时间复杂度 O(n log n)

## 二、查找算法

### 二分查找
- 有序数组查找
- 时间复杂度 O(log n)
- 前提：有序数组

### 线性查找
- 遍历查找
- 时间复杂度 O(n)

## 三、基础算法思想

### 递归
- 函数自身调用
- 关键：base case（递归终止条件）
- 注意递归深度

### 双指针
- 滑动窗口：处理子数组/子串问题
- 快慢指针：检测链表环、找到中点

### 贪心算法
- 局部最优解
- 不一定能得到全局最优

### 动态规划
- 保存子问题结果
- 空间换时间
- 典型：斐波那契、背包问题

## 四、复杂度分析

### 时间复杂度
- O(1) 常数时间
- O(log n) 对数时间
- O(n) 线性时间
- O(n log n) 线性对数时间
- O(n²) 平方时间

### 空间复杂度
- 算法占用的额外内存空间
- 递归调用栈空间也算
