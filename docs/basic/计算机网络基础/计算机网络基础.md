# 计算机网络基础

## 一、TCP/IP 协议栈

### 四层模型
1. 应用层（HTTP、FTP、DNS）
2. 传输层（TCP、UDP）
3. 网络层（IP、ICMP）
4. 链路层（Ethernet）

### 三次握手
1. 客户端 → 服务器：SYN
2. 服务器 → 客户端：SYN + ACK
3. 客户端 → 服务器：ACK

### 四次挥手
1. 客户端 → 服务器：FIN
2. 服务器 → 客户端：ACK
3. 服务器 → 客户端：FIN
4. 客户端 → 服务器：ACK

## 二、HTTP/HTTPS

### HTTP 方法
- GET：获取资源
- POST：提交数据
- PUT：更新资源
- DELETE：删除资源

### 状态码
- 2xx：成功（200 OK）
- 3xx：重定向（301 永久重定向）
- 4xx：客户端错误（404 Not Found）
- 5xx：服务器错误（500 Internal Error）

### 请求头/响应头
- Content-Type、Content-Length
- Cookie、Set-Cookie
- Authorization

## 三、Cookie / Session / Token

- **Cookie**：存储在客户端的文本信息
- **Session**：服务端存储的会话信息
- **Token**：无状态的令牌（如 JWT）

## 四、DNS 与跨域

- **DNS**：域名解析为 IP 地址
- **跨域**：浏览器同源策略限制，CORS 是解决方案
