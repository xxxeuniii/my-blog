# 多模态提示

图片+文字组合输入的提示方式。

---

## 支持多模态的模型

| 模型 | 支持格式 |
|------|----------|
| GPT-4 Vision | 图片URL/Base64 |
| Claude-3 | 图片+文档 |
| Gemini | 图片+视频+音频 |
| 通义千问VL | 图片 |

---

## 示例一：设计稿分析

```
你是一名UI设计师，请分析以下设计稿：

[图片]

请从以下维度分析：
1. 整体布局是否合理
2. 色彩搭配是否协调
3. 交互体验可能存在的问题
4. 改进建议

返回JSON格式：
{
  "layout_score": 分数1-10,
  "color_score": 分数1-10,
  "issues": ["问题列表"],
  "suggestions": ["改进建议"]
}
```

## 提示技巧

| 技巧 | 说明 |
|------|------|
| 指定格式 | 明确告诉模型你希望的输出格式 |
| 分步引导 | 先识别图片类型，再分析内容 |
| 补充文字 | 图片质量差时，用文字补充说明 |
| 控制数量 | 单次不要发太多图，建议1-3张 |

## 常见问题

| 问题 | 解决方案 |
|------|----------|
| 图片太大 | 压缩到1-2MB |
| 模型不识别 | 用文字描述关键内容 |
| 输出不准确 | 指定输出格式，减少自由度 |
