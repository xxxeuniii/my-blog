# Top-p (核采样)

另一种控制输出随机性的参数，与Temperature类似但不同。

---

## 参数解释

```
Top-p = 0.0 ~ 1.0（通常）

Top-p = 1.0：
- 使用全部词汇概率分布
- 输出最随机、最大胆

Top-p = 0.9：
- 只从前90%概率的词汇中选择
- 过滤掉长尾、低概率词汇

Top-p = 0.1：
- 只从最高概率的10%词汇中选择
- 输出非常保守、稳定
```

---

## Top-p vs Temperature

| 参数 | 控制方式 | 效果 |
|------|----------|------|
| Temperature | 直接缩放概率分布 | 控制整体随机性 |
| Top-p | 累积概率阈值 | 动态调整词汇候选集 |

```
举例：
词汇概率：[0.5, 0.2, 0.15, 0.1, 0.05]

Top-p=0.9：
累积：0.5 → 0.7 → 0.85 → 0.95
包含到0.9，所以前4个词都在候选集

Top-p=0.5：
累积：0.5 → 0.7
只包含前2个词
```

---

## 推荐用法

| 场景 | Top-p | Temperature |
|------|-------|-------------|
| 稳定输出 | 0.9 | 0.3 |
| 平衡模式 | 0.95 | 0.7 |
| 创意模式 | 1.0 | 1.0 |

---

## 实际建议

```
一般规则：
1. 只调一个（Temperature 或 Top-p），不要同时调
2. 默认从 Top-p=0.95 或 Temperature=0.7 开始
3. 太保守 → 降低Top-p 或 提高Temperature
4. 太随机 → 提高Top-p 或 降低Temperature

安全区间：
- 精确任务：Top-p 0.8-0.95 或 Temperature 0.1-0.3
- 创意任务：Top-p 0.95-1.0 或 Temperature 0.8-1.0
```
