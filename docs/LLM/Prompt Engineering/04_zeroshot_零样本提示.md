# Zero-shot (零样本提示)

不给例子，直接下达任务。

---

## 示例一：翻译

```
将以下中文翻译成英文：
"人工智能正在改变我们的生活方式"
```

---

## 示例二：分类

```
将以下评论分类为：正面 / 负面 / 中性
"这家餐厅的上菜速度太慢了，等了一个小时"
```

---

## 示例三：总结

```
总结以下文章的核心观点：
"今天，全球科技巨头齐聚深圳，参加年度创新大会。会上展示了最新的人工智能技术，涵盖医疗、教育、制造业等多个领域。与会专家表示，AI技术将在未来5年内渗透到日常生活的方方面面。"
```

---

## 适用场景

| 场景 | 说明 |
|------|------|
| 简单明确任务 | 翻译、润色、纠错等 |
| 通用能力强 | 模型本身就能做得好的任务 |
| 快速尝试 | 先试zero-shot，不行再加few-shot |

## 核心要点

- 最简单直接的方式
- 适合常见、通用任务
- 复杂任务可能效果不佳
- 可作为基线，不满意再升级到few-shot

## vs Few-shot

| 方式 | 何时用 |
|------|--------|
| Zero-shot | 简单任务、模型能力强 |
| Few-shot | 复杂格式、特定风格、罕见任务 |
