Tailwind CSS жҢҮеҚ— вҖӢ

д»Җд№ҲжҳҜ Tailwind CSS вҖӢ

Tailwind CSS жҳҜдёҖдёӘе®һз”Ёдјҳе…Ҳзҡ„ CSS жЎҶжһ¶пјҢйҖҡиҝҮжҸҗдҫӣеӨ§йҮҸзҡ„ utility class жқҘеҝ«йҖҹжһ„е»әиҮӘе®ҡд№үз•ҢйқўгҖӮ

е®үиЈ…дёҺй…ҚзҪ® вҖӢ

1. е®үиЈ…дҫқиө– вҖӢ

bash
npm install tailwindcss @tailwindcss/vite

2. Vite й…ҚзҪ® вҖӢ

javascript
// vite.config.js
import tailwindcss from '@tailwindcss/vite'

export default {
  plugins: [tailwindcss()]
}

3. еҲӣе»әж ·ејҸж–Үд»¶ вҖӢ

css
/* src/style.css */
@tailwind base;
@tailwind components;
@tailwind utilities;

ж ёеҝғжҰӮеҝө вҖӢ

1. Utility First вҖӢ

html
<!-- дҪҝз”Ё Tailwind utility class -->
<div class="flex items-center justify-center px-4 py-2 bg-blue-500 text-white rounded-lg">
  Button
</div>

2. е“Қеә”ејҸи®ҫи®Ў вҖӢ

html
<!-- дёҚеҗҢеұҸе№•е°әеҜёеә”з”ЁдёҚеҗҢж ·ејҸ -->
<div class="text-center sm:text-left md:text-right">
  Responsive Text
</div>

3. иҮӘе®ҡд№үдё»йўҳ вҖӢ
