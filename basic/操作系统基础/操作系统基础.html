操作系统基础 ​

一、进程、线程、协程 ​

进程 ​

  • 程序的一次执行实例
  • 系统资源分配的基本单位
  • 独立内存空间，进程间通信需要 IPC

线程 ​

  • CPU 调度的基本单位
  • 同一进程内线程共享内存
  • 比进程更轻量

协程 ​

  • 用户态轻量级线程，由程序自身控制
  • 切换不需要内核态，开销极小
  • 适合 I/O 密集型任务

二、并发与并行 ​

  • 并发：多个任务交替执行，宏观同时
  • 并行：真正同时执行，需多核 CPU
  • 同步：任务按顺序执行
  • 异步：任务不必等待，可继续执行

三、内存管理 ​

栈内存 ​

  • 存放函数调用、局部变量
  • 自动分配回收，地址连续
  • 空间小（1-8MB），线程独有

堆内存 ​

  • 存放动态分配对象
  • 需手动或 GC 回收
  • 空间大，地址不连续，线程共享

ps -ef    # 查看进程
kill -9   # 终止进程
netstat   # 查看端口
