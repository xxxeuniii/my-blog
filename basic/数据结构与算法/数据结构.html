数据结构 ​

一、线性结构 ​

数组（Array） ​

  • 连续内存存储
  • 随机访问 O(1)
  • 插入/删除 O(n)

链表（Linked List） ​

  • 节点通过指针连接
  • 插入/删除 O(1)
  • 访问 O(n)
  • 单链表、双链表、循环链表

栈（Stack） ​

  • 先进后出（LIFO）
  • 操作：push（入栈）、pop（出栈）
  • 应用：函数调用栈、括号匹配

队列（Queue） ​

  • 先进先出（FIFO）
  • 操作：enqueue（入队）、dequeue（出队）
  • 优先队列、循环队列

二、散列结构 ​

哈希表（Hash Table） ​

  • 键值对存储
  • 查找 O(1)
  • 冲突解决：链地址法、开放地址法
  • JavaScript 中对象和 Map 就是哈希表

三、树形结构 ​

二叉树 ​

  • 每个节点最多两个子节点
  • 遍历方式：
    • 前序：根 → 左 → 右
    • 中序：左 → 根 → 右
    • 后序：左 → 右 → 根
    • 层序：按层次遍历

二叉搜索树（BST） ​

  • 左子树所有节点小于根节点
  • 右子树所有节点大于根节点
  • 查找、插入、删除 O(log n)

平衡树 ​

  • AVL 树、红黑树
  • 自动平衡，保证操作 O(log n)

四、图结构 ​

图（Graph） ​

  • 顶点（Vertex）和边（Edge）
  • 有向图 / 无向图
  • 邻接矩阵、邻接表存储

常用算法 ​

  • BFS（广度优先搜索）
  • DFS（深度优先搜索）
  • Dijkstra 最短路径
